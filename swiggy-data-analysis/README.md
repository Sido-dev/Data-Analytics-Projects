# 🍔 Swiggy Data Analysis Project

## 📌 Overview

This project focuses on analyzing Swiggy restaurant data to uncover meaningful insights related to customer preferences, pricing trends, ratings, and popular cuisines. The goal is to transform raw data into actionable insights using data analysis and visualization techniques.

---

## 🎯 Objectives

* Analyze restaurant data to identify trends in ratings and pricing
* Discover the most popular cities and cuisines
* Understand the relationship between price and customer ratings
* Generate insights that can help improve business decisions

---

## 🛠️ Tools & Technologies

* **Python**
* **Pandas** – Data manipulation
* **NumPy** – Numerical operations
* **Matplotlib & Seaborn & plotly express** – Data visualization
* **Jupyter Notebook** – Analysis and reporting

---

---

## 🔍 Data Analysis Process

1. **Data Collection** – Load dataset from Excel file
2. **Data Cleaning** – Handle missing values and duplicates
3. **Exploratory Data Analysis (EDA)**

   * Distribution of ratings
   * City-wise restaurant analysis
   * Price trends
4. **Visualization** – Graphical representation using plots
5. **Insights Generation** – Extract meaningful conclusions

---

## 📊 Key Insights

* Most restaurants fall within a moderate price range
* Ratings are generally concentrated between 3.5 and 4.5
* Certain cities have a significantly higher number of restaurants
* Higher prices do not always guarantee better ratings
